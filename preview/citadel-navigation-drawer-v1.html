Dynamic Preview: Navigation Drawer

Citadel Burger Menu Drawer

This page tests the actual Citadel navigation drawer module in isolation. Resize to mobile width and test opening, closing, Escape key, focus handling and backdrop behaviour.

Preview only. This page is marked noindex and exists only to test the drawer navigation module before production rollout.

Mobile-first

Citadel of Kang must treat mobile compatibility as a first-class requirement.

Accessible

The drawer updates aria-expanded, supports Escape close and keeps focus inside the drawer while open.

Reusable

The module is route-agnostic and uses data attributes, not Chambers-only assumptions.

Scroll Test

This extra content confirms that page scrolling is restored after the drawer closes.

Backdrop Test

Opening the menu should show a dim backdrop. Clicking the backdrop should close the drawer.

Focus Test

Keyboard focus should move into the drawer and return after closing.
