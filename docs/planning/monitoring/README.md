# Legal Monitoring Working Directory

Last updated: **13 July 2026**

This directory is reserved for current, time-bounded legal monitoring outputs generated under:

```text
docs/planning/LEGAL_MONITORING_AND_ORIGINAL_ARTICLE_WORKFLOW.md
docs/planning/LEGAL_MONITORING_OUTPUT_TEMPLATE.md
docs/planning/LEGAL_SOURCE_WATCHLIST.md
```

Use the naming pattern:

```text
YYYY-MM-DD_LEGAL_MONITORING_BATCH.md
```

After a monitoring batch is completed and any resulting article/status work is closed, move the dated output to:

```text
docs/archive/monitoring-snapshots/YYYY-MM/
```

Do not treat archived monitoring notes as current legal authority. Re-verify time-sensitive law from primary sources before publication.
